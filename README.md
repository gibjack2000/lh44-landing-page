# LH44 — Lewis Hamilton Tribute Landing Page

A cinematic premium landing page for 7x Formula 1 World Champion Lewis Hamilton.

## Features
- 120-frame canvas animation in Hero & Quote sections
- Animated stats counter (Breaking Records section)  
- Responsive gallery grid
- Smooth scroll-triggered reveal animations

## Deployment
Drop the contents of this folder into any static hosting provider:
- GitHub Pages
- Netlify (drag & drop)
- Vercel
